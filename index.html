Operations

With Python, you can do any basic operations we usually do with a calculator machine

For example, you can do this 5 + 2 and Python will give the result: 7.

Some others examples:

  • 52 - 4 Python give us :48
  • 60 + 50 Python give us :110
  • 50 / 2 Python give us :25
  • 60 % 8 Python give us :4( because 60 = (8 * 6) + 4
  • 10 * 2 Python give us :20
  • 10 ** 2 Python give us :100( 100 squared )
Print() function

The print() function is very easy to use. It prints in the Python console something you choose to print.
For example, if I code print(10), Python will print in the console 10.
I can also print some text: print("Hello World"), Python will print in the console Hello World.
